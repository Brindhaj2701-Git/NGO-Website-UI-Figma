# 🌍 NGO Website UI Design (Figma)

## 📌 Project Overview
This project is an NGO Website UI prototype designed using **Figma** as part of my internship at **InAmigos Foundation**.

The objective of this project was to design a clean, user-friendly, and visually appealing NGO website that highlights the organization's mission and makes it easy for users to donate, volunteer, and learn about the foundation's impact.

---

## ✨ Features
- 💚 Hero Section
- 🎁 Donation Section
- 🤝 Volunteer Registration Form
- 📊 Impact Showcase
- 📢 Campaigns Section
- 💬 Testimonials
- 📞 Contact & Footer

---

## 🛠️ Tools Used
- Figma
- UI/UX Design Principles
